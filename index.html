Powerful Features for Dedicated Lexicographers

Database Project Setup

Define your dictionary's structure from scratch, including custom fixed fields, SQL data types, and widget types (Line Edit or Text Edit).

Dual-Panel Workflow

Edit entries in the form while simultaneously browsing the paginated, alphabetized Headword Index in a dedicated side panel for superior efficiency.

Repeatable Fields

Easily manage multiple meanings, examples, or references for a single headword using flexible, dedicated input widgets that grow with your entry.

Smart Search & Filter

Execute full-text searches across all fields (headword, meaning, pos) and utilize advanced filtering by Part of Speech (POS) with smart auto-completion.

Data Integrity Tools

Ensure quality control with built-in utilities to check for duplicate Headwords and Meanings, alongside simple database backup and restore functionality.

Flexible Import/Export

Seamlessly migrate your data. Import and export your entire dictionary data effortlessly using standard CSV, JSON, and publish-ready PDF formats.

Built for Lexical Precision
